## Project

I want to create a platform for our Profit Coaches that is going to be deployed on Vercel. Right now I don't think we need any memory; it's just single-state things that they can access and get feedback on, like their messaging.

The first thing I want to build is a LinkedIn funnel analyzer. They can put in the numbers they’ve got across six boxes, the app compares them to target KPIs, highlights the constraints, and gives a set of actions.

Design: really simple, clean, modern, accent color blue, slightly off-white background with white cards.

## Spec

They should be able to enter numbers for:

- sent connections or connection requests
- connected
- replied
- interested
- booked calls
- clients

KPIs:

- sent to connected: 15%
- connected to replied: 20%
- replied to interested: 5%
- interested to booked: 50%
- booked to client: 25%

As they put their numbers in, it should highlight whether conversion rates are green, yellow, or red.
